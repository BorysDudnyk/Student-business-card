# Student Business Card

Simple multi-page static website created as a university web development assignment.  
The project simulates a personal student profile card with navigation between sections.

## Features
- Multi-page structure (HTML navigation)
- Personal profile layout
- Sidebar with academic information
- Responsive-like fixed layout design
- CSS styling with reset
- Image header with overlay effect
- Basic UI/UX navigation system

## Architecture
- Pure HTML5
- CSS3 styling
- Multi-page static structure
- No frameworks / no JavaScript

## Pages
- Home (About me)
- University info (KNTU)
- Tech & Soft Skills
- Additional information page
